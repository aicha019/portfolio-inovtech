Bienvenue sur le projet Inov'Tech

Dans le cadre de notre formation au CNAM en DevOps, nous avons réalisé ce projet en groupe. Nous avons imaginé un stylo révolutionnaire, le Translator'Pen, capable de traduire instantanément plus de 60 langues, afin de faciliter la communication internationale et encourager l’apprentissage des langues.

Le groupe

Ce projet a été réalisé par : Aïcha FOFANA, Meriem MEZDARI et Codou Diouf FAYE.

Répartition du travail :
- inovtech.html + CSS : Meriem MEZDARI
- formulaire.html + CSS : Codou Diouf FAYE
- page-produit.html + CSS : Aïcha FOFANA
- panier.html + CSS : toutes les 3
La conception des pages et la charte graphique ont été réalisées par l’ensemble du groupe.
